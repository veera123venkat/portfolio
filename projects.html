Smart Step Locking System in Bus

This project enhances passenger safety by automatically locking the bus door when it starts moving and unlocking it when it stops. Built using:

  • Arduino Uno
  • ESP32 Module (Wi-Fi enabled)
  • Relay Channel
  • LEDs, Buzzer
  • Capacitors and Resistors

Purpose: Prevent passengers from falling while boarding or getting down during motion.
